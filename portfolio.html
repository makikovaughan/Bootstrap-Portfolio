Project 5 LIRI Bot




Project Description

This program is Liri Bot. LIRI is a _Language_ Interpretation and Recognition Interface. LIRI will be a command line node app that takes in parameters and gives you back data.

Project Details

  • On a command line, type node liri.js concert-this artist name and hit enter.
  • On a command line, type node liri.js spotifty-this-song song title and hit enter.
  • On a command line, type node liri.js movie-this movie title and hit enter.
  • On a command line, type node liri.js do-what-it-says and hit enter. Then, the file(random.txt) will be read and return the result.
